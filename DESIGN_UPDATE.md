## 🎨 Customization Options

If you want to customize further:

### **Change Colors:**
Edit `style.css` and replace:
- `#003366` - Primary navy blue
- `#FFD700` - Gold accent
- `#f5f7fa` - Background gray

### **Change Fonts:**
Edit `style.css`:
- Headings: Change `'Georgia'`
- Body: Change `'Arial'`

### **Change Logo Size:**
Edit `style.css`:
```css
.nav-logo {
  height: 60px;  /* Change this */
}

.login-logo img {
  height: 100px;  /* Change this */
}
```

### **Change College Name:**
Edit HTML files and replace:
```html
Massachusetts Institute of Technology
```
